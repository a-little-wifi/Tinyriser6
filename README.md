# Tinyriser6
Tinyriser for the Tiny6 generation of Lenovo Tiny PCs
Same idea as the original Tinyriser, but for the next gen.  
Compatible with M90q gen 1 and P340 Tiny (M70q and M80q don't have the riser slot populated)

Work in progress
